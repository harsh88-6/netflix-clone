## Netflix Landing Page Clone

This project is a responsive Netflix landing page clone built using HTML, CSS, and JavaScript.  
The goal of this project was to practice frontend development skills and recreate a modern UI inspired by Netflix.

### Features
- Responsive design for all screen sizes  
- Clean and modern UI  
- Styled sections similar to the original Netflix landing page  

### Tech Stack
- HTML  
- CSS  


### Purpose
This project was created for learning and practice purposes only. It is not affiliated with Netflix.
